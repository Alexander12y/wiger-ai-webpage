# Video de Fondo para Hero Section

## Archivo requerido:
- **Nombre:** `hero-background.mp4`
- **Ubicación:** `/public/videos/hero-background.mp4`

## Especificaciones recomendadas:
- **Resolución:** 1920x1080 (Full HD) o superior
- **Formato:** MP4 (H.264)
- **Duración:** 10-30 segundos (se reproduce en loop)
- **Tamaño:** Preferiblemente menos de 10MB para optimizar la carga
- **Calidad:** Alta calidad pero optimizado para web

## Contenido sugerido:
- Imágenes relacionadas con manufactura e industria
- Fábricas, líneas de producción, tecnología
- Movimientos suaves y profesionales
- Colores que complementen la marca (azules, grises)

## Nota:
El video se reproduce automáticamente en loop sin sonido. Si el archivo no está presente, se mostrará un fondo gradiente como fallback.
